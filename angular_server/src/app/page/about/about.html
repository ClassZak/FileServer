7. Работа с файлами и папками

{{'\tРаздел «Файлы» предназначен для просмотра, поиска и управления файловой системой сервера. По умолчанию открывается корневая директория.'}}

{{'\tПользователь видит только те файлы и папки, доступ к которым разрешён его учётной записи. Файловая система организована следующим образом: в корневой папке находится папка «groups», внутри которой находятся папки всех групп. Каждый член группы имеет право на создание, чтение, изменение и удаление файлов в своей группе. Пользователь может состоять в нескольких группах. Администратор имеет полный доступ ко всем файлам.'}}

{{'\tНа странице присутствуют следующие элементы:'}}

  • {{'\tФорма поиска (позволяет искать файлы в текущей директории).'}}
  • {{'\tПанель навигации в файловой системе (хлебные крошки).'}}
  • {{'\tТаблица файлов (если есть доступные файлы).'}}
  • {{'\tТаблица папок (если есть доступные папки).'}}
  • {{'\tКнопка «Создать папку».'}}
  • {{'\tКнопка «Загрузить файл» (для загрузки файла на сервер).'}}

7.1. Панель навигации по каталогам (Breadcrumbs)

{{'\tВ верхней части страницы расположена панель навигации по каталогам, отображающая текущий путь в файловой системе.'}}

  • {{'\tКнопка «Корень» — переход в корневой каталог.'}}
  • {{'\tКнопка «Назад» — возврат на один уровень выше.'}}
  • {{'\tКлик по элементу пути — переход в выбранный каталог.'}}

7.2. Таблица папок

{{'\tТаблица папок отображает список каталогов, доступных в текущем разделе файловой системы.'}}

  • {{'\tКнопка «Удалить» — удаление папки (доступно только администратору).'}}

7.3. Таблица файлов

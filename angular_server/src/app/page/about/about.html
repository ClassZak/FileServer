5. Работа с файлами и папками

{{'\tРаздел «Файлы» предназначен для просмотра, поиска и управления файловой системой сервера.'}}

{{'\tПользователь видит только те файлы и папки, доступ к которым разрешён его учётной записи.'}}

5.1. Панель навигации по каталогам (Breadcrumbs)

{{'\tВ верхней части страницы расположена панель навигации по каталогам (хлебные крошки), отображающая текущий путь в файловой системе.'}}

  • {{'\tКнопка «Корень» — переход в корневой каталог файловой системы.'}}
  • {{'\tКнопка «Назад» — возврат на один уровень выше.'}}
  • {{'\tКлик по элементу пути — переход в выбранный каталог.'}}

5.2. Поиск файлов

{{'\tПоле поиска предназначено для поиска файлов по имени в текущем каталоге и вложенных папках.'}}

  • {{'\tПоле ввода — ввод части имени файла.'}}
  • {{'\tКнопка «Поиск» — запуск процедуры поиска.'}}

{{'\tРезультаты поиска отображаются в отдельных таблицах файлов и папок.'}}

5.3. Таблица папок

{{'\tТаблица папок отображает список каталогов, доступных в текущем разделе файловой системы.'}}

  • {{'\tИмя папки.'}}
  • {{'\tПолный путь.'}}
  • {{'\tДата последнего изменения.'}}

{{'\tДля каждой папки доступны следующие действия:'}}

  • {{'\tОткрыть — переход внутрь выбранной папки.'}}
  • {{'\tУдалить — удаление папки (доступно только администратору).'}}

5.4. Таблица файлов

{{'\tТаблица файлов отображает список файлов текущего каталога или результатов поиска.'}}

  • {{'\tИмя файла.'}}
  • {{'\tРасширение файла.'}}
  • {{'\tРазмер файла.'}}
  • {{'\tДата последнего изменения.'}}

{{'\tДля каждого файла доступны следующие действия:'}}

  • {{'\tСкачать — загрузка файла на устройство пользователя.'}}
  • {{'\tУдалить — удаление файла (доступно только администратору).'}}

5.5. Модальные окна раздела «Файлы»

Подтверждение удаления файла

  • {{'\tКнопка «Подтвердить» — окончательное удаление файла.'}}
  • {{'\tКнопка «Отмена» — закрытие окна без выполнения действия.'}}

Подтверждение удаления папки
